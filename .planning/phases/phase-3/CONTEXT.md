## 任务包分解

### WBS-3.1: 资产搜索模块（2 人周）

1. 搜索引擎集成（Elasticsearch）
2. 模糊查询与精确查询
3. 高级筛选（业务域、主题域）
4. 搜索结果展示

### WBS-3.2: 表级血缘查询模块（3 人周）

1. 完整血缘图查询算法
2. 层级展开与收缩
3. 多视图切换
4. 血缘图导出

### WBS-3.3: 字段级最小链路模块（4 人周）

1. 最短路径算法（Dijkstra/A*）
2. 字段血缘提取
3. 表达式解析与展示
4. 多源汇聚处理

### WBS-3.4: 问题排查页（2 人周）

1. 血缘与运行态关联
2. 最近批次展示
3. 变更信息展示
4. 快速跳转

## 验收标准

- ✅ 功能测试通过率 ≥ 95%
- ✅ 表级血缘查询 P95 ≤ 3 秒
