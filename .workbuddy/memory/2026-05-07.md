## 2026-05-07 UI 浅色主题改造

将数据血缘分析系统的所有可视化页面从深色主题（#0a0e27 背景）全面改为浅色主题：

### 修改的文件
- `query_template.html` — 查询版页面模板
- `generate_query_html.py` — LAYER_CONFIG 配色
- `generate_layered_html.py` — 分层版生成脚本（含内嵌CSS/JS）
- `generate_enhanced_html.py` — 增强版生成脚本（含内嵌CSS/JS）
- `generate_fixed_html.py` — 修复版生成脚本（含内嵌CSS/JS）
