# 数据血缘分析系统 - 测试计划

## 一、系统概述

数据血缘分析系统是一个基于 FastAPI 的数据血缘追溯平台，支持：
- 文件解析（.tab/.prc）
- 字段级血缘查询
- 指标口径追溯
- 指标血缘分析

## 二、测试范围

### 2.1 功能模块

| 模块 | 主要功能 | 测试优先级 |
|------|---------|-----------|
| 解析层 | 文件上传、解析任务管理 | 高 |
| 血缘展示层 | 表搜索、字段血缘查询、图谱展示 | 高 |
| 指标口径 | 口径查询、Pipeline 展示 | 高 |
| 指标血缘 | 指标搜索、指标血缘追溯 | 高 |

### 2.2 测试类型

- **后端单元测试**：API 接口、服务逻辑
- **前端 E2E 测试**：用户交互流程
- **数据准确性测试**：验证血缘数据的正确性
